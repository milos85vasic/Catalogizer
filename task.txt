Extend the project with the following application: installation wizard for machines that will host the core. Desktop application which works by the configuration wizard principle. User can scan the smb network, browse it and pick roots or subdirecotries of smb shares and add them as the sources to the configuration - json we use. It is possible to open existing json config and modify it by adding, editing or removing the smb sources. Once you finish make sure that it has 100% tests coverage and that everything is documented.
