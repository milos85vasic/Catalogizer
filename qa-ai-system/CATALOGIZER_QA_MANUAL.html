🔧 Troubleshooting Guide

❓ Common Issues & Solutions

API Server Connection Issues

Problem: Cannot connect to Go API server
Solution: Ensure Go is installed and API server is running on correct port

# Check Go installation
go version

# Start API server manually
cd ../catalog-api
go run main.go

Android Build Failures

Problem: Android APK build fails
Solution: Verify Android SDK and build tools are properly configured

# Check Android SDK
echo $ANDROID_HOME

# Manual build test
cd ../catalogizer-android
./gradlew assembleDebug

Database Connection Issues

Problem: Database schema validation fails
Solution: Verify database configuration and run migrations

# Check database status
sqlite3 ../database/catalogizer.db ".tables"

# Run migrations
cd ../catalog-api
go run migrate.go
