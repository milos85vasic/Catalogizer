# Video Course

Catalogizer offers comprehensive video training courses for users, administrators, and developers.

## User Course (6 Modules)

Complete training from installation through advanced media management.

| Module | Title | Duration | Topics |
|--------|-------|----------|--------|
| 1 | Introduction & Installation | 45 min | Overview, system requirements, container/manual install |
| 2 | Getting Started | 75 min | Web UI, storage sources, browsing, search, analytics |
| 3 | Advanced Media Features | 75 min | Favorites, collections, playlists, subtitles, conversion |
| 4 | Multi-Platform Experience | 55 min | Android, Android TV, Desktop, API client |
| 5 | Administration | 65 min | User management, security, monitoring, backup |
| 6 | Developer Guide | 70 min | Architecture, dev setup, adding features, submodules |

**Total Duration**: ~6 hours 25 minutes

## Advanced Modules

| Module | Title | Duration | Topics |
|--------|-------|----------|--------|
| 7 | Testing & Quality Assurance | 60 min | Go tests, Vitest, Playwright, challenges, security testing |
| 8 | Deployment & Production | 55 min | Container builds, Podman, production config, monitoring |

## Certification Levels

- **Catalogizer User** - Complete Modules 1-4 with 80% assessment score
- **Catalogizer Administrator** - Complete Modules 1-5 with 80% assessment score
- **Catalogizer Developer** - Complete all 6 core modules with 80% assessment score
- **Catalogizer Expert** - Complete all 8 modules including advanced with 80% assessment score

## Course Materials

Each module includes:
- Narrated video scripts with timestamps
- Slide presentations with speaker notes
- Hands-on exercises with step-by-step instructions
- Assessment quizzes with answer keys
- Reference code examples from the actual codebase
