## About Catalogizer

Catalogizer is a modern media collection management system that automatically detects, categorizes, and organizes your media files across multiple storage protocols including SMB, FTP, NFS, WebDAV, and local filesystem.

### Key Features

- **Multi-Protocol Support**: Access media on SMB, FTP, NFS, WebDAV, and local filesystems
- **Real-time Monitoring**: Automatic detection and organization of new media
- **Cross-Platform**: Works on Windows, macOS, Linux, Android, and Android TV
- **Modern UI**: Responsive web interface with real-time updates
- **API-First**: RESTful API for integration with other systems
- **Open Source**: Licensed under MIT - contribute to the project
